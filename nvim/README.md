# Neovim Setup

## For 0.11

See `:help news-features`. Some highlights so I remember:

* `grn` now renames.
* `grr` goes to LSP references.
* `gri` goes to LSP impls.
* `gra` does code actions in normal mode.
* `[d` and `]d` move to diagnostics.
* Omni complete (`<C-xC-o>`) supports LSP completion now.
